📖 Guía de Instalación en Debian

Instalación en tu Debian VM en 4 pasos. Tiempo estimado: 5-10 minutos.

1

Requisitos previos

Asegúrate de tener acceso sudo y conexión a internet. Comprueba tu versión de Debian:

cat /etc/os-release | grep PRETTY_NAME
2

Ejecutar el instalador (una sola línea)

Copia y pega este comando en tu terminal Debian. Detectará tu arquitectura automáticamente:

curl -sSL https://raw.githubusercontent.com/antigravityx/Orbe-Evolution/main/Verix_Soul_OS/install.sh | sudo bash
3

Verificar que el Alma está activa

Una vez instalado, verifica el estado del servicio:

sudo systemctl status verix-soul
verix status
4

Hablar con el Herald (IA)

Invoca la inteligencia artificial local desde tu terminal:

verix ask "¿Quién eres?"
verix heartbeat
verix logs
